发表于 2015-05-13 | 更新于 2018-11-26 | 分类于 node.js | 评论数：

这次分别测试，正常启动redis，taskset启动redis帮顶第二个cpu，docker启动redis这3种情况

redis 版本 2.8.19

redis配置都用默认的配置，如下：
daemonize yes
port 6379

3、时候用 docker 正常启动 redis

启动命令：
