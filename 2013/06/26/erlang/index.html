erlang安装

发表于 2013-06-26 | 更新于 2018-11-26 | 分类于 erlang | 评论数：

windows下安装：

linux下安装，亲测centos6 64bit
step 1:
添加Erlang Solutions key 支持
rpm –import http://binaries.erlang-solutions.com/debian/erlang_solutions.asc

step 2:
将erlang的repo文件添加到/ete/yum.repos.d/下
wget http://binaries.erlang-solutions.com/rpm/centos/erlang_solutions.repo
mv erlang_solutions.repo /etc/yum.repos.d/

step 3:
添加 RPMforge 支持 (64位)
wget http://packages.sw.be/rpmforge-release/rpmforge-release-0.5.2-2.el6.rf.x86_64.rpm

导入 key
rpm –import http://apt.sw.be/RPM-GPG-KEY.dag.txt

安装 RPMforge（64位）
rpm -i rpmforge-release-0.5.2-2.el6.rf.*.rpm

step 4:
更新并安装
yum update
yum install esl-erlang
如果报python版本问题的话，手动连以下yum的python版本即可

step 5:
确认安装成功
打开erl
erl
hello world
io:format(“hello world ~n”).

参考：
1.http://www.erlang-solutions.com/section/132/download-erlang-otp
2.http://wiki.centos.org/AdditionalResources/Repositories/RPMForge#head-f0c3ecee3dbb407e4eed79a56ec0ae92d1398e01

亲测centos5.5 32bit：
1、安装openssl
wget http://www.openssl.org/source/openssl-1.0.1e.tar.gz
tar -zxvf http://www.openssl.org/source/openssl-1.0.1e.tar.gz
cd openssl-1.0.1e
./config
make && make install

2、安装依赖

yum install build-essential m4

yum install openssl

yum install openssl-devel

yum install unixODBC

yum install unixODBC-devel

yum -y install make gcc gcc-c++ kernel-devel m4 ncurses-devel openssl-devel

3、下载erlang
wget http://www.erlang.org/download/otp_src_R15B01.tar.gz
tar -zxvf otp_src_R15B01.tar.gz
cd otp_src_R15B01
./configure –prefix=/usr/local/erlang –enable-hipe –enable-threads –enable-smp-support –enable-kernel-poll
make && make install

4、配置环境变量
vi /etc/profile

ERL_HOME=/usr/local/erlang
PATH=$ERL_HOME/bin:$PATH
export ERL_HOME PATH
source /etc/profile
更新环境变量

5、执行erl，进入erlang的shell成功安装

# erlang
